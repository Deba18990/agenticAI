# Basic React Login / Logout App

A minimal React app scaffolded with Vite. It includes a simple login screen and a logout screen using client-side state.

## Quick Start

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open the local URL shown by Vite in your browser.
